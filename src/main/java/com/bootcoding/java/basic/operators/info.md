
## Operators in Java


| Symbols | Arithmetic Operators          |
|:---:    | :---:                         |
|---------|-------------------------------|
| `+`     | add                           |
| `-`     | substract                     |
| `*`     | multiply                      |
| `/`     | divide                        |
| `%`     | modulus division              |
| `++`    | increment by 1 (pre and post) |
| `--`    | decrement by 1 (pre and post) |
| `%`     | modulus division              |
